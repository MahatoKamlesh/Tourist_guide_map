About the Interactive Tourist Guide Map

This application provides an interactive map for tourists to explore key attractions, weather updates, and nearby facilities.

Team Members

  • Team Member : Kamlesh Kumar Mahato, Kanchan Adhikari, Pusp Raj Ojha

Features

  • Interactive map with multiple layers.
  • Weather updates on map click.
  • Proximity analysis for nearby POIs.
