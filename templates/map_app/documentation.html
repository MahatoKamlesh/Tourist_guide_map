Technical Documentation

Project Objectives

The aim is to provide an interactive map with real-time weather updates and proximity analysis for tourists.

Technology Stack

  • Frontend: HTML, CSS, JavaScript, Leaflet.js
  • Backend: Django
  • Database: PostgreSQL with PostGIS
  • APIs: OpenWeather API, Buffer analysis

System Components

  • Interactive Map: Displays multiple base and overlay layers.
  • Weather Updates: Fetches and displays weather information.
  • Buffer Analysis: Finds nearby points of interest.

User Documentation

Follow these steps to use the application:

  1. Open the map from the home page.
  2. Enable weather or proximity analysis using the toggles.
  3. Click on the map to fetch information.
  4. Explore additional information on the About page.
